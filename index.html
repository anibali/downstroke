_.get_function (func_name)
Returns the function associated with name func_name (or nil if no such function exists).

If func_name is a function, it is returned unchanged.

Parameters:

  • func_name

Usage:

     local add = _.get_function('+')
 assert_true(add(40, 2) == 42)

_.carve (collection, portions)
Carves up collection into portioned groups. Any uneven remaining elements are placed in the last group.

Parameters:

  • collection
  • portions

Usage:

     local carved = _.carve({1, 2, 3, 4, 5}, {0.6, 0.2, 0.2})
 assert_true(carved == {{1, 2, 3}, {4}, {5}})

_.assign (dict, ...)
Merges properties from sources into dict. The rightmost value of a property takes precedence.

Mutates dict.

Parameters:

  • dict
  • ...

Usage:
